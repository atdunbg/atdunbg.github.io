tabs note

名称 用法
class 【可选】标识，不同的标识有不同的配色（ default / primary / success / info / warning / danger ）
no-icon 【可选】不显示icon
style 【可选】可以覆盖配置中的 style（simple/modern/flat/disabled）

实例

note info simple
提示块标签
endnote

note info modern
提示块标签
endnote

note info flat
提示块标签
endnote

note info disabled
提示块标签
endnote

名称 用法
color 【可选】顔色(default / blue / pink / red / purple / orange / green)
icon 【可选】可配置自定义 icon (只支持 fontawesome 图标,也可以配置 no-icon )
style 【可选】可以覆盖配置中的 style（simple/modern/flat/disabled）

实例

note pink ‘fas fa-car-crash’ simple
小心开车 安全至上
endnote

note red ‘fas fa-fan’ modern
这是三片呢？还是四片？
endnote

note green ‘fab fa-internet-explorer’ flat
前端最讨厌的浏览器
endnote

note orange ‘fas fa-battery-half’ disabled
你是刷 Visa 还是 UnionPay
endnote

note blue no-icon
2023年快到了….
endnote

Button

使用方法：

名称 用法
url 链接
text 按钮文字
icon [可选] 图标
color [可选] 按钮背景顔色(默认style时），按钮字体和边框顔色(outline时)default/blue/pink/red/purple/orange/green
style [可选] 按钮样式 默认实心 outline/留空
layout [可选] 按钮布局 默认line block/留空
position [可选] 按钮位置 默认最左边（前提设置了layout为block） 默认最左边 center/right/留空
size [可选] 按钮大小 large/留空
